# Data_Driven_Water_Quality_Surveilance_And_Loss_Detection_System
An IoT and AI-powered water quality surveillance and loss detection platform designed for real-time monitoring of water distribution systems using sensor-driven analytics, anomaly detection, and predictive insights to identify contamination, leakage, flow irregularities, and resource losses.


-----------------------------IMPORTANT-----------------------------

1) environment.yml file run command: " conda env create -f environment.yml "

2) activate venv command: " conda activate data-driven-water-quality-surveillance "
